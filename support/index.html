Contact

お問い合わせ

用途に合わせて窓口を選んでください。下の連絡先欄を差し替えるだけで公開できます。

アプリの不具合

Yamanote Runner

問い合わせ先

ここに Yamanote Runner の問い合わせ先を入れてください。
例: support@yamanoterunner.example または GitHub Issues の URL

問い合わせに含める内容

アプリ名、端末名、OS バージョン、再現手順、発生日時を添えてください。

運用全般

個人の窓口

問い合わせ先

ここにあなた個人の問い合わせ先を入れてください。
例: your-name@example.com

使い分け

アプリの不具合は左の窓口、運用全般はこの窓口、という形で分けられます。

この2つの欄を書き換えれば、そのまま公開できます。アプリが増えたらカードを複製して増やせます。

Overview

アプリ概要

対応アプリは、iPhone の健康記録・位置情報・課金機能などを、それぞれの用途に合わせて使います。このページは共通の問い合わせ窓口として運用し、アプリごとの補足は必要に応じて追加します。

Billing

サブスクと購入復元

購入後に表示が反映されない場合は、アプリ内の「購入を復元する」をお試しください。

解決しない場合は、App Store にサインイン中の Apple ID を確認のうえ、上の窓口へ状況をお送りください。
