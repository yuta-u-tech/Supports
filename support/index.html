共有サポートページ

複数アプリで共通運用する使い方、購入復元、問い合わせ方法の案内です。

お問い合わせ

不具合報告やご要望は、公開先が決まり次第、この共有ポータルの GitHub Issues で受け付けます。問い合わせ時は、アプリ名、端末、OS バージョン、再現手順を添えてください。

公開後に Issue URL を差し込みます。

アプリ概要

対応アプリは、iPhone の健康記録、位置情報、課金機能などをそれぞれの用途に合わせて使います。

このページは共通の問い合わせ窓口として運用し、アプリごとの補足は必要に応じて追加します。

サブスクと購入復元

サブスクリプションの購入後に表示が反映されない場合は、アプリ内の「購入を復元する」を試してください。

それでも解決しない場合は、App Store にサインインしている Apple ID を確認したうえで、公開後の Issue から状況を送ってください。
